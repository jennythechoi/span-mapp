# span-mapp

## Project setup
```
npm install
```

### Set up environment
Create a file titled .env.local
Paste this variable declaration into it
```
VUE_APP_MAPBOX_API_ACCESS_TOKEN=
```
Paste a Mapbox Public API token following the equals, one is provided when you create an account.
